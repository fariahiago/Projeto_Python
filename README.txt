Esse projeto será o desafio de aprendizagem de utilização do Python em ambiente Linux, bem como tentativas de sincronizar com banco de dados e com servidores de rede.

Primeiro passo: Codificar o projeto
Segundo passo: Integrar com banco de dados
Terceiro passo: Fazer deploy em servidor

1.1: Criar sistema de gerenciamento de usuários (1/N) Meta: finalizar até 5/10

Relato dia 1: Criei objeto USER, que vai receber atributos e as funções de permissão

1.2 Criar a função e as permissões que cada user terá assim como as funções.
    Permissões Conferente: Lançar quantidade de entrada e saida de produtos do estoque.	   Atributos: Produto, Quantidade, NF de movimentação.
    Permissões Vendedor: Lançar venda, alterar valor dentro da margem estabelecida pelo    gerente, consultar estoque;
    Permissões gerente: Gerar relatório de vendas individual, cadastrar usuários, autor    desconto maior que margem para vendedor.
    Permissões cliente: Verificar estoque e verificar preço.
